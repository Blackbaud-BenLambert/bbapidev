An object representing the email address to add.
Property Type Description
constituent_id string Required. The constituent identifier.
type string Required. The email address type, from the Email Address Types table.
address string The email address.
do_not_email boolean Indicates whether or not the constituent wishes to be contacted at this email.
primary boolean Indicates whether or not this is the primary email record for a constituent.
inactive string Indicates whether or not this email is active.
