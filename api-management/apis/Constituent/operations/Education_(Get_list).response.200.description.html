Returned when the operation is successful. The response body contains a collection of education entities.
Property Type Description
id string The education identifier.
constituent_id string The constituent identifier.
school string The school name.
type string The type of education.
degree string The degree received.
class_of string The graduating year.
date_entered Fuzzy date The first date of attendance.
date_graduated Fuzzy date The graduation date.
status string The status of the education.
primary boolean The primary alumni flag.
major string The major course of study.
minor string The minor course of study.
