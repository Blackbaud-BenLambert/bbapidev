An object representing the action to add.
Property Type Description
constituent_id* string The constituent identifier.
category* string The action category.
type string The action type.
date* date The action date.
summary string The action summary.
description string The action description.
completed boolean Indicates whether or not the action has been completed - note that this value will be set based on the action status value if the system has been configured to use custom action statuses. Available values are true or false. If no value is specified, a value of false is assumed.
completed_date date Indicates the date on which the action was completed.
status string The action status (only applicable if the system has been configured to use custom action statuses)
fundraisers array of ActionFundraiser The set of fundraisers associated with the action
proposal_id string The identifier of the proposal associated with the action.
